# 📝 Task Manager App

A simple Task Manager application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ➕ Add tasks
- 📅 Set due date
- 🔥 Set priority (High / Low)
- ✅ Mark tasks as completed
- ✏️ Edit tasks
- 🗑 Delete tasks
- 💻 Responsive design

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
